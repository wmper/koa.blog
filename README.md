### koa.blog

一个由 `node+koa+vue` 搭建的简单 blog，存储使用 mysql

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev || npm run dev--watch
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run pm2
```
